# Task

Write a handoff document summarising the current conversation so a fresh agent can continue the
work. Save to the agent scratchpad directory. Do not duplicate content already captured in other
artifacts, reference them by path or URL instead. Redact any sensitive information, such as API
keys, passwords, or personally identifiable information.
